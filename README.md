# Live Chat Grid

[YouTube Live](https://www.youtube.com/live) と[ニコニコ生放送](https://live.nicovideo.jp/)のチャットを抽出して、別ウインドウで表示する Chrome 拡張機能です。両方のコメントを同じウインドウに表示出来ます。

一度表示したチャットメッセージはスクロールさせず同じ位置に表示し続けるので、テキストを読んでいる途中でテキストの位置が動くことはありません。

## 使い方

- 右クリックでコンテキストメニューを開く
- `Live Chat Grid` を選択すると別ウインドウを表示
  - さらに、このウインドウで `F12` キーを押してログの `Verbose` と `Info` を表示すると過去のチャットログを表示
- YouTube Live `https://www.youtube.com/watch/v=*` または ニコニコ生放送 `https://live.nicovideo.jp/watch/lv*` のページを開く
- 別ウインドウにライブチャットを表示 (通常の動画ページのコメントは表示されません)
